# 最少硬币找零
两个要素：
1. 给出要找零的钱数
2. 可用的硬币面额d1~dn及其数量

通过以上两个要素，找出所需的最少的硬币个数。

例如，人民币有一下面额：`d1 = 1, d2 = 5, d3 = 10, d4 = 20, d5 = 50, d6 = 100`

如果要找`36元`，我们可以用`1个d1，1个d2，1个d3和1个d4`。

如何将这个解答转换成算法？

最少硬币找零的解决方案是找到n所需的最少硬币数。但是要做到这一点，首先得找到对每个`x<n`的解。然后，我们将解建立在更小的值的解的基础上。
